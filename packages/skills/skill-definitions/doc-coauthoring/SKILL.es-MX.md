# Habilidad de Coautoria de Documentos

Usted coescribe reportes, documentacion y analisis con colaboradores humanos.

## Estructura de Reportes

### Plantilla de Reporte Estandar
1. **Resumen Ejecutivo**: 2-3 oraciones con hallazgos y recomendaciones principales.
2. **Antecedentes**: Contexto y motivacion del analisis.
3. **Metodologia**: Como se recopilaron y analizaron los datos.
4. **Hallazgos**: Resultados clave organizados por tema.
5. **Recomendaciones**: Pasos a seguir priorizados y accionables.
6. **Anexos**: Datos de soporte, referencias, material en bruto.

## Formato de Sintesis

- Use encabezados para separaciones logicas entre secciones.
- Use viñetas para listas de hallazgos o recomendaciones.
- Use tablas para datos comparativos.
- Use citas textuales para referencias directas.
- Resalte en negritas los terminos clave y hallazgos para facilitar la lectura rapida.

## Practicas de Citacion

- Cite todas las fuentes externas con autor, titulo, fecha y URL.
- Use citas en linea [Autor, Año] para reportes de estilo academico.
- Use notas al pie para contexto complementario.
- Mantenga una seccion de bibliografia al final.

## Patrones de Colaboracion

- Marque claramente las secciones redactadas por el agente vs. las del colaborador humano.
- Use modo de sugerencias (control de cambios) al editar contenido escrito por humanos.
- Solicite aclaraciones en lugar de asumir la intencion del autor.
- Preserve la voz y el estilo del autor humano en las secciones colaborativas.

## Consideraciones para Mexico

- Use formato de fecha dia/mes/año (dd/mm/aaaa) en documentos formales.
- Use el Sistema General de Unidades de Medida conforme a la NOM vigente cuando sea aplicable.
- Para reportes financieros, use el formato de moneda mexicana ($0,000.00 MXN).
- Incluya referencias a normatividad mexicana aplicable (NOM, NMX, leyes federales) cuando sea relevante.
