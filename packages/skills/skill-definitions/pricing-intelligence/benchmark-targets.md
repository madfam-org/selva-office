## Meta notes

- Only add a URL here if you've checked it renders prices without an
  account. Pages that gate pricing behind "contact sales" are a bucket
  we should not benchmark against unless we have a real sales motion.
- Keep the list under ~6 URLs per product. The weekly run is a signal,
  not a census.
- When a URL moves, log the change in this file — don't silently let
  the scraper break and report stale data.
